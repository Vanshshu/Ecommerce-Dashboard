# Ecommerce-Dashboard
This project focuses on building an interactive dashboard to track and analyze online sales data. The dashboard helps in understanding sales performance and provides meaningful insights through visual representation.

🔹 Key Features
Created an interactive dashboard to monitor and analyze online sales data
Implemented complex parameters to enable drill-down functionality across worksheets
Applied filters and slicers for better customization and user interaction
Established data connections and joined multiple tables for data modeling
Performed calculations to manipulate data and support user-driven parameters
Designed various customized visualizations including:
Bar Chart
Pie Chart
Donut Chart
Clustered Bar Chart
Scatter Chart
Line Chart
Area Chart
Map
Slicers
🔹 Objective

The main objective of this project is to provide a dynamic and user-friendly dashboard that allows users to explore sales data efficiently and gain actionable insights.
